# Gerador de Senhas Seguras

Este projeto foi desenvolvido utilizando HTML, CSS e JavaScript. O objetivo é permitir que o usuário personalize e gere uma senha segura de forma simples e intuitiva.

O usuário pode escolher a quantidade de caracteres da senha e selecionar quais tipos de caracteres deseja incluir, como letras maiúsculas, letras minúsculas, números e símbolos. Com base nessas opções, o sistema gera automaticamente uma senha aleatória.

Além disso, o projeto avalia a força da senha por meio de uma barra indicadora e estima quanto tempo um computador levaria para descobrir essa senha utilizando um ataque de força bruta.

## Funcionalidades

- Gerar senhas aleatórias.
- Aumentar ou diminuir a quantidade de caracteres da senha.
- Escolher entre letras maiúsculas, minúsculas, números e símbolos.
- Exibir a força da senha em uma barra de progresso.
- Calcular o tempo estimado para descobrir a senha.
- Interface simples, organizada e de fácil utilização.

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

## Como Executar

1. Faça o download ou clone este repositório.
2. Abra a pasta do projeto.
3. Execute o arquivo `index.html` em qualquer navegador.

Não é necessário instalar bibliotecas ou programas adicionais.

## Objetivo

Este projeto foi desenvolvido com o objetivo de praticar os conceitos de HTML, CSS e JavaScript, além de demonstrar a importância da criação de senhas fortes para aumentar a segurança digital.
